# izDog

Supply a dog breed to see if it's known.

Results supplied by [Dog API](https://dog.ceo/dog-api/).

## Usage

### Good (woof!)

```
node index.js "dachshund"
```

```
node index.js "westhighland terrier"
```

### Bad (yelp!)

```
node index.js "cumberland"
```

```
node index.js "bratwurst"
```
